# tonibot
Toni, the Ginsporter. It has gin inside and flees if someone gets close. 

## About
Tonibot was constructed in advance of 36c3 CCC congress. 

It consists of "hacked" hoverboard, a raspberry and a octosonarX2 I2C device.
